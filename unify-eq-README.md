# UNIFY Equalizer Bass Booster — v3 Community + SEO GitHub Pages package

All public files use the `unify-eq-` prefix and assets live inside `unify-eq-assets/`, so this package can share a repository with UNIFY Music.

## Upload
Upload the contents of this folder to the repository root. Keep the existing UNIFY Music files. The new Community image is `unify-eq-assets/images/unify-eq-community-presets.webp`.

## After deployment
1. Submit `https://unifyplayer.online/unify-eq-sitemap.xml` in Google Search Console.
2. Inspect `https://unifyplayer.online/unify-eq-home-page.html` and request indexing.
3. Run Google Rich Results Test for the deployed homepage.
4. Keep the Search Console DNS TXT verification record in Hostinger.
