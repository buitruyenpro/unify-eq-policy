# UNIFY Music — prefixed GitHub Pages package

This package is designed for a shared GitHub Pages repository that contains pages for multiple apps.
Every UNIFY Music file uses the `unify-music-` prefix, and all assets live inside `unify-music-assets/` to avoid filename collisions.

Upload these files and the `unify-music-assets` folder to the root of the repository.

## Public URLs

- Home: `https://unifyplayer.online/unify-music-home-page.html`
- Privacy Policy: `https://unifyplayer.online/unify-music-privacy.html`
- Terms of Service: `https://unifyplayer.online/unify-music-terms.html`
- Account & Data Deletion: `https://unifyplayer.online/unify-music-delete-data.html`
- Optional sitemap: `https://unifyplayer.online/unify-music-sitemap.xml`

## Google Auth Platform

- Application home page: `https://unifyplayer.online/unify-music-home-page.html`
- Privacy policy: `https://unifyplayer.online/unify-music-privacy.html`
- Terms of service: `https://unifyplayer.online/unify-music-terms.html`

Do not overwrite the repository's existing `index.html` or global `robots.txt` unless you intentionally want UNIFY Music to become the root website.

## Homepage hero

The first-screen hero does not use a promotional screenshot. It is rendered in the browser with HTML Canvas and CSS using harmonic wave equations, a radial spectrum, Lissajous curves, deterministic particles and a responsive player interface. Promotional images remain available only in the homepage gallery.
